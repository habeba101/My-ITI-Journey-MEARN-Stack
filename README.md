# My-ITI-Journey-MEARN-Stack

This repository documents my learning journey at ITI as a MEARN Stack Developer.

## Overview

This repository is organized into topic-focused labs and exercises for web development, JavaScript, front-end technologies, and related programming concepts. Each top-level folder groups one or more labs or projects by subject.

## Project Documentation

Below is a summary of each project area in this repository.

### Advanced JS

Contains advanced JavaScript labs that explore objects, DOM manipulation, event handling, and modular code patterns.

- `Lab 1` covers object programming and use of JavaScript objects.
- `Lab 2` and `Lab 3` include more interactive UI exercises using buttons and separate module files.

### Client Side Technologies

A collection of front-end labs focused on HTML, CSS, and JavaScript integration.

- `Lab 1` through `Lab 7` include exercises in page structure, styling, DOM scripting, event-driven behavior, and small web application features.
- Many labs include separate script files for assignment sections and interactive components.

### CSS Preprocessor

A SASS-based project demonstrating CSS preprocessor usage.

- Includes `main.scss`, `main.css`, and `Profile.html`.
- Organized with SASS partials under `abstracts`, `base`, `components`, `layout`, `themes`, `utilites`, and `vendors`.

### Data Structure

JavaScript implementations of common data structures and algorithms.

- `Lab 1` includes algorithm problems like search and math operations.
- `Lab 2` has linked list implementations such as `DoubleLinkedList.js` and `LinkedList.js`.
- `Lab 3` covers basic algorithms and structures like `Fibonacci.js`, `Queue.js`, and `Stack.js`.
- `Lab 4` focuses on tree and binary search algorithms with `BinaryTree.js` and `BinarySearch.js`.
- `Lab 5` includes sorting algorithm implementations in `Sorting.js`.

### ECMA Script 6

Exercises in modern JavaScript and ES6 features.

- `Lab 1` includes assignments using `Assignment.html` and `Assignment.js`.
- `Lab 2` contains multiple ES6-focused parts demonstrating syntax, arrow functions, classes, and other new language features.
- `Lab 3` includes AJAX exercises with `AJAX.html`, `AJAX.js`, and sample JSON data.

### Generative AI

Projects and experiments related to AI and algorithm comparison.

- `Lab 2` includes code like `comparing.js`, performance testing in `performance_timeit.py`, and a `QuickSort.js` demo.
- `Lab 3` contains further AI-related lab work.

### GraphQL

GraphQL learning labs.

- `Lab 1` and `Lab 2` contain exercises to learn GraphQL query and schema concepts.

### HTML 5 & CSS3

Introductory and intermediate HTML/CSS labs.

- `Lab 1`, `Lab 2`, and `Lab 3` include HTML pages and styling exercises that build web page structure and responsive design basics.

### JavaScript Design Patterns

Labs covering design pattern concepts in JavaScript.

- `Lab 1` and `Lab 2` demonstrate reusable patterns for organizing JavaScript applications.

### jQuery

jQuery-based DOM manipulation and event handling labs.

- `Lab 1` and `Lab 2` include examples of selecting elements, responding to user input, and updating page content with jQuery.

### Responsive Web Design

Responsive layout and adaptive interface labs.

- `Lab 1`, `Lab 2`, and `Lab 3` focus on building pages that adapt to different screen sizes and devices.

### TypeScript

TypeScript exercises that introduce typed JavaScript development.

- `Lab 1` and `Lab 2` include TypeScript lab files and exercises on static typing and modern JavaScript workflows.

## How to Use

Open any folder and run the HTML files in a browser or inspect the JavaScript files to follow the exercise logic. For SASS projects, view the compiled CSS or edit `.scss` source files.
